Shop Management System

A professional desktop-based Shop Management System developed using C# WinForms and SQL Server.  
This application helps manage products, employees, sales, and invoices efficiently through a user-friendly interface.

Key Features
- Secure Login System
- Admin & Employee Management
- Product Management
- Selling & Billing System
- Invoice Generation
- Profile & Password Management

Technologies Used
- C#
- Windows Forms (.NET Framework)
- SQL Server
- ADO.NET

Getting Started
1. Open the project in Visual Studio  
2. Configure the SQL Server database  
3. Update the connection string in `App.config`  
4. Build and run the application  

Project Purpose
This project was developed for academic and learning purposes to demonstrate desktop application development and database management using C#.
